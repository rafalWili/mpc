# 🎨 custom-bootstrap

Mały projekt pomocniczy do kompilacji własnych stylów SCSS w motywie WordPress (Child Theme). Używa `sass` i `yarn` w wersji 4.x.


## 🚀 Szybki start

### 1. Instalacja zależności

```bash
yarn install

```

###  Tryb obserwacji (watch)
```bash
yarn watch:css
```

###  Reczna kompilacja 
```bash
yarn build
```
